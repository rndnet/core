RnDnet server installation by helm 

Install
-------

1. Copy files 
    - common
    - values.yaml
    - install-dry-test.sh
    - install.sh
  from `install-skel` to this base directory.

2. Modify them with your settings.

3. Run  ./install-dry-test.sh for check configs

4. Run ./install.sh 

Uninstall
---------

Run ./uninstall.sh
